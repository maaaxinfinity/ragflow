请帮我解释一下React的虚拟滚动是如何工作的？

虚拟滚动是一种性能优化技术，主要用于渲染大型列表。它只渲染可见区域的DOM元素，而不是整个列表。

核心原理：
  1. 计算可视区域高度
  2. 确定可见项的索引范围
  3. 只渲染可见项 + 预渲染项
  4. 动态调整容器偏移量

在RAGFlow的FreeChat中，我们使用 @tanstack/react-virtual 实现了消息列表的虚拟滚动，可以流畅渲染1000+条消息。

技术实现

import { useVirtualizer } from '@tanstack/react-virtual';

const rowVirtualizer = useVirtualizer({
  count: messages.length,
  getScrollElement: () => parentRef.current,
  estimateSize: () => 150, // 估计高度
  overscan: 5, // 预渲染5条
});
