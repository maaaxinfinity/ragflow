## 🎯 使用指南

### 新成员加入

1. **快速了解项目**:
   ```
   .todo/PROGRESS.md → 查看当前进度
   .todo/TODO.md → 查看任务清单
   ```

2. **深入了解技术方案**:
   ```
   .todo/FREECHAT_IMPROVEMENT_PLAN.md → 完整方案
   .todo/FREECHAT_IMPROVEMENTS_SUMMARY.md → 实施总结
   ```

3. **开始开发**:
   ```
   web/src/pages/free-chat/README.md → 开发指南
   .memory/04-freechat-improvements.md → 改进记录
   ```

### 任务管理

1. **查看待办任务**:
   ```bash
   cat .todo/TODO.md | grep "^\- \[ \]"
   ```

2. **查看当前阶段**:
   ```bash
   cat .todo/PROGRESS.md | grep "当前阶段"
   ```

3. **更新任务状态**:
   - 编辑 `TODO.md`，将 `- [ ]` 改为 `- [x]`
   - 更新 `PROGRESS.md` 中的进度百分比
   - 记录完成时间和交付物

### 进度汇报

**周报内容来源**:
- `PROGRESS.md` - 整体进度和质量指标
- `TODO.md` - 本周完成的任务
- `PROGRESS.md` - 风险和问题部分

**月报内容来源**:
- `PROGRESS.md` - 里程碑达成情况
- `TODO.md` - 各阶段完成统计
- `PROGRESS.md` - 团队贡献统计

## 🔄 文档更新流程

### 任务完成时

1. 更新 `TODO.md`:
   - 将任务状态改为 `[x]`
   - 填写完成时间
   - 列出交付物

2. 更新 `PROGRESS.md`:
   - 更新阶段进度百分比
   - 更新任务统计
   - 添加到"最近完成"列表
   - 更新质量指标
   - 记录变更日志

3. 如果是阶段完成:
   - 创建阶段完成报告（如 `FREECHAT_PHASE2_COMPLETE.md`）
   - 更新里程碑状态

### 新问题或风险

1. 更新 `PROGRESS.md`:
   - 添加到"当前风险"表格
   - 设置级别（🔴高/🟡中/🟢低）
   - 描述影响和应对措施

2. 问题解决后:
   - 从"当前风险"移到"已解决问题"
   - 记录解决方案和时间
